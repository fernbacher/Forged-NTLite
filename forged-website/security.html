Security & Trade-Offs

Forged disables a lot of Windows components for performance and privacy. Here's exactly what's changed and what you're trading away. If you need real-time antivirus, Forged is not for you.

Windows Defender — Fully Annihilated

  • Services disabled (Start=4)
  • TamperProtection=0, DisableAntiSpyware=1
  • Scheduled task re-kills Defender processes
  • Boot-time safety task prevents reactivation
  • IFEO blocks on MsMpEng.exe and related binaries
  • MpCmdRun.exe renamed

Trade-off: No real-time malware protection. You're responsible for what you run. Fine for a gaming-only install where you only launch Steam and games.

Microsoft Edge — Completely Removed

  • Browser, updater, and DevTools removed via XOS RemoveEdge
  • Uses region spoofing to bypass Microsoft's uninstall restrictions
  • WebView2 Runtime preserved for app compatibility

Trade-off: No built-in browser. Bring your own on a USB drive. Apps that embed Edge WebView still work — the runtime is left intact.

UAC — Disabled

EnableLUA=0. Everything runs as admin. No elevation prompts.

Trade-off: Any process can make system-level changes without asking. Convenient on a machine you control, dangerous if you run untrusted software.

Windows Update — Paused Until 2038

  • Update service paused with a far-future date
  • WSUS server nulled
  • Driver updates blocked via registry
  • Scheduled tasks for automatic maintenance disabled

Trade-off: No security patches, no driver updates, no feature updates. Manual update checks may partially revert some tweaks. You're freezing Windows in its current state.

VBS / HVCI — Toggle Scripts on Desktop

Two scripts on the desktop: VBS-ON.bat and VBS-OFF.bat.

  • VBS OFF (default): Better performance, lower latency.
  • VBS ON: Required for any kernel anti-cheat that doesn't work on Linux (Valorant, FACEIT, etc.). Exception: League of Legends doesn't need it. Not required on Windows 10.

Telemetry & Privacy

  • All telemetry services disabled
  • Cortana, Bing integration, web search removed
  • Advertising ID disabled
  • Activity history and clipboard sync disabled
  • Diagnostic data set to minimum

Network Optimizations

  • Nagle's algorithm disabled
  • Network throttling index maxed
  • DNS cache increased
  • TCP autotuning set to normal

Bottom line: Forged is a "get in, play your game, get out" Windows install. Stripped down for maximum gaming performance with zero regard for security best practices. Use it for what it's designed for: running kernel anti-cheat games that won't work on Linux.
