Dual-Boot with Limine

Forged is meant to live alongside your Linux install. Windows will hijack the boot order after install. Here's how to add it to Limine so you can pick at boot.

Before You Install

  1. Shrink your Linux partition or leave unallocated space. Use gparted or whatever your distro ships.
  2. Leave at least 60 GB for Windows. More if you're installing big games.

Installation

  1. Boot from the Forged USB drive.
  2. When asked where to install, point it at the unallocated space. The installer won't touch your Linux partitions unless you explicitly tell it to.
  3. autounattend.xml handles the rest. Fully automated.

Adding Windows to Limine

First, find where Windows put its bootloader on your EFI partition:

sudo ls /mnt/esp/EFI/Microsoft/Boot/

You should see bootmgfw.efi. Now add an entry to Limine:

sudo tee -a /boot/limine.conf <<EOF

/Windows 10
    protocol: efi
    path: boot():/EFI/Microsoft/Boot/bootmgfw.efi
EOF

Windows UTC Clock

Windows stores local time in the hardware clock by default. Linux uses UTC. This causes clock drift every time you switch.

Forged sets RealTimeIsUniversal=1 in the registry so Windows uses UTC. Clocks stay in sync, no extra config.

Already handled. The post-install script applies this during Phase 4. You don't need to touch anything.

Picking an OS at Boot

Limine will show both entries on startup. Pick whichever you need. Both OSes stay independent — Windows won't mess with your Linux bootloader after this.
