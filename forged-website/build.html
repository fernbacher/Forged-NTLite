Build Guide

You need a Linux machine and a Windows 11/10 ISO. Grab one from Microsoft or massgrave.dev. Then run a single command.

Prerequisites

Install the build dependencies:

# Arch Linux / CachyOS
sudo pacman -S p7zip wimlib libisoburn python3

# Debian / Ubuntu
sudo apt update
sudo apt install --needed p7zip-full wimtools xorriso python3

# Fedora
sudo dnf install -y p7zip wimlib-utils xorriso python3

Clone the Repository

git clone https://github.com/fernbacher/Forged.git
cd Forged

Build the ISO

Point the script at your Windows ISO:

./forge-iso.sh /path/to/en-us_windows_11_consumer_editions_version_26h1_*.iso

The script does everything in one shot:

  1. Extracts the ISO with 7z
  2. Finds the Windows 11/10 Pro index, exports only that edition
  3. Mounts the WIM with wimlib-imagex
  4. Injects all payload files and strips AppX bloat offline
  5. Commits changes, builds a UEFI+BIOS hybrid ISO with xorrisofs
Build time: 5–10 minutes, depending on storage speed. Fully automated.

Output

You'll get Forged-Win11.iso (or Forged-Win10.iso) in the Forged/ directory. Flash it with Ventoy, Rufus, or dd.

Browsers are not included. Forged removes Edge completely. Bring your own browser installer on your USB drive.

Post-Install

The ISO includes autounattend.xml — it bypasses TPM/SecureBoot, creates a local admin account, skips OOBE privacy screens, and auto-launches the Forged script on first login. Let it run.

Recommendations Once You're on Forged

  • Install dotNetFx35 on Windows 10
  • Use Alt App Installer to install NVIDIA Control Panel without the Microsoft Store
  • Install VC Redist and DirectX
  • Install Open-Shell on Windows 11 if you want a classic start menu
  • Run ms-gamebar-annoyance.bat to kill the Game Bar pop-up
